To install run the install script in an empty directory.
Make sure you give it execution. 
Ex: <br /> <br />

in terminal:
mkdir gesture_detection <br />
cd gesture_detection <br />
cp ~/Downloads/install.sh ./ <br />
chmod +e install.sh <br />
./install <br /> <br />
